# Bảng Tin Tài Chính (demo)

Trang tĩnh HTML — GitHub Pages.

## Bật GitHub Pages

1. Trên GitHub: repo **Settings** → **Pages** (menu trái).
2. **Build and deployment** → **Source**: **Deploy from a branch**.
3. **Branch**: `main`, folder **`/ (root)`** → **Save**.
4. Sau vài phút, site tại: **https://quanluu.github.io/demo/**

*(Nếu dùng tên user/repo khác, thay `QuanLuu` / `demo` cho đúng.)*
